    マップオブジェクト配置

    MFC 版 DlgAdminMapObject のワークフローを Web で再現する試験実装です。マップを選択すると、サーバーから読み込んだ実データに基づくオブジェクト一覧と配置状況を確認できます。

    マップを選択し、テーブルの行をクリックして配置を選択するか、グリッドをクリックして座標を指定します。

    配置ID オブジェクトID 名称 X Y 属性 当たり判定
    マップ情報を読み込み中です。

    配置プレビューと編集

    座標グリッドをクリックすると、該当セルに新規オブジェクトを配置できます。既存オブジェクトを選択すると詳細を編集できます。

    アイテム種別一覧（プレースホルダー）

    アイテム分類を管理する画面です。

    • カテゴリツリー表示
    • 種別追加・編集フォーム
    • 関連マスタの参照

    アイテム一覧（プレースホルダー）

    マップ上のアイテムを検索・生成する機能を想定しています。

    • マップ/カテゴリフィルター
    • アイテム生成リクエスト
    • 削除・移動操作の監査ログ

    武器一覧（プレースホルダー）

    武器パラメータを確認する画面です。

    • 武器カテゴリ・レアリティでのソート
    • ステータス比較ビュー
    • エクスポート機能

    噴出し一覧（プレースホルダー）

    メッセージ吹き出し定義を管理する画面を予定しています。

    • 吹き出しテンプレのプレビュー
    • テキスト/スタイル編集
    • プレビュー再生

    エフェクト一覧（プレースホルダー）

    エフェクトパラメータを調整する画面です。

    • エフェクト検索とタグ管理
    • アニメーションプレビュー
    • タイムライン編集

    監査レポートダウンロード（プレースホルダー）

    Wave 3 で予定している監査ログ出力画面です。

    • 期間指定と CSV/JSON エクスポート
    • フィルター（操作種別、担当者）
    • 自動配信スケジュール設定

    操作履歴検索（プレースホルダー）

    管理操作の履歴を検索・照会する画面です。

    • クエリビルダー形式のフィルター
    • 結果テーブルと詳細パネル
    • エクスポート・共有リンク生成

    フェイルバック手順（プレースホルダー）

    Web UI 障害時に MFC へ切り戻す手順をナビゲートする画面です。

    • 現在のシステム状態チェック
    • 切り戻し手順ステップ表示
    • 復旧チェックリスト

    初期ステータス設定（プレースホルダー）

    新規キャラクターの初期パラメータを編集する画面です。

    • テンプレート選択と複製
    • 各種ステータス値の編集フォーム
    • バリデーション結果の表示
