About the game

個人制作だからこそ、ちょっと不思議なオンライン世界。

なつかしいドット絵のフィールドを歩き、出会った人と話し、アイテムを集める。大作とは違う手触りを残したまま、長い年月をかけて少しずつ育てています。

Explore the world

小さな世界の、大きな寄り道。

誰かとすれ違ったり、木陰で立ち止まったり。効率よりも、つい寄り道したくなる空気を大切にしています。
