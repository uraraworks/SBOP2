# Claude Code Instructions

このファイルは公開リポジトリに含める、Claude Code 向けの共通プロジェクト指示です。
個人設定やローカル環境固有の運用は `CLAUDE.local.md` に書き、コミットしないでください。

## Language

- 回答・作業説明・エラーメッセージの解説は日本語で行う。
- commit message も日本語で作成する。
- コードコメントや README などのドキュメントも、既存方針に合わせて日本語を優先する。

## Build Policy

- 修正後の確認では、ユーザーが明示的に求めた場合を除き、フルビルドよりも関連ファイル・関連プロジェクトに絞った確認を優先する。
- フルビルドは時間がかかるため、必要性を説明してから実行する。
- ビルドやテストのログは全文を会話に流さず、ログファイルに保存して必要箇所だけ確認する。

## Log Handling

- 成功時は終了コードや短い成功報告だけにし、長いログを読み戻さない。
- 失敗時だけ、`error` / `failed` / `warning` などの重要行と末尾の短い範囲を抽出して確認する。
- 既存の巨大ログファイルも全体を開かず、検索や末尾表示で必要範囲に絞る。

## File Search

- 探索はまず `rg` / `rg --files` などの高速な検索を使う。
- 大きいファイルは全体を読まず、必要な範囲だけ確認する。
- 広範囲の調査では、結果を要約してコンテキストを増やしすぎない。
